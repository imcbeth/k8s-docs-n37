# Monitoring Overview

Comprehensive monitoring setup with Prometheus, Grafana, and additional tools.

## Current Stack

- **Prometheus**: Metrics collection and storage
- **Grafana**: Visualization and dashboards  
- **UniFi Poller**: Network monitoring
- **Node Exporter**: Hardware metrics
- **Synology SNMP**: NAS monitoring (planned)
